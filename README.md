### Features

- **50+ Individual Recipe Files** - Each recipe is its own file for easy editing and linking
- **Rich Metadata** - Frontmatter includes prep time, cook time, nutrition, tags, and more
- **Category Organization** - Recipes organized into logical folders
- **Cross-linking** - Recipes can reference related recipes
- **Tagging System** - Find recipes by tags like #easy, #quick, #comfort-food
- **Search** - Full-text search across all recipes
